# GameDemo_Wraith
This is a 3D Roguelike game demo based on Unreal Engine 4.27.2.
2022年暑期参加游戏公司实训营，和几个小伙伴一起用UE 4.27.2 制作了此游戏demo，以蓝图为主完成了游戏完整流程，打包了可游玩版本。后来对游戏开发有了更多理解，2025年开始陆续进行了一些个人更新，并于2026年3月创建此仓库记录。
